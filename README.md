# Weasley App API Server

## Runbook

JVM Option 

```cmd
-Dspring.profiles.active=local
```

## Spec

- Spring boot 2.6.1
- Jvm version 11 ~ 13 
- kotlin
- JPA & Hibernate 
- gradle
- h2 (local)
- postgresql (prod)
- aws
